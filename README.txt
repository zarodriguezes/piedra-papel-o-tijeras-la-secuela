# piedra-papel-o-tijeras
juego de piedra papel o tijeras 

Cómo se juega Piedra, Papel o Tijeras?
Es un juego clásico de 2 jugadores, donde cada uno elige una de tres opciones en secreto
 y luego la revelan al mismo tiempo. Las reglas para decidir quién gana son simples.

 Opciones:
Piedra 
Papel 
Tijeras 

 Reglas del juego:
Piedra gana a Tijeras 
Tijeras gana a Papel 
Papel gana a Piedra 
Si los dos eligen lo mismo: Empate!

En  Python:
El programa te pide que escribas: piedra, papel o tijeras.
Luego la computadora elige al azar una de las tres.
El programa compara ambas elecciones y te dice si ganaste, perdiste o empataste.

Si es multijugador te pedira 2 nombres uno de cada jugador 
jugador 1 elige y se esconde su opcion
jugador 2 elige 
El programa compara ambas elecciones y te dice si ganaste, perdiste o empataste.

Version de python usada
Ultima version 3.13.5
